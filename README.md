# TaLK-Corpus

A regionally diverse **Sri Lankan Tamil speech dataset** created to support research in speech and language technology. This repository hosts the **audio-only public release** of the **TaLK-Corpus**.

---

## ✨ Overview

TaLK-Corpus is a carefully curated Sri Lankan Tamil speech dataset designed to support research in:

- **Automatic Speech Recognition (ASR)**
- **Dialect and accent analysis**
- **Low-resource language technology**
- **Speech processing and benchmarking**

The dataset covers speech from **22 administrative districts of Sri Lanka**, making it useful for studying regional variation in Sri Lankan Tamil.

This repository contains **only the audio files**.

---

## 🎧 Key Features

- **Language:** Sri Lankan Tamil  
- **District Coverage:** 22 districts of Sri Lanka  
- **Speakers:** 22 native speakers  
- **Total Duration:** Approximately 1 hour and 33 minutes  
- **Total Utterances:** 1,214  
- **Audio Format:** WAV  
- **Sampling Rate:** 16 kHz  
- **Channels:** Mono  
- **File Naming:** Anonymised file names

## 🔒 Privacy and Ethics

All participants provided informed consent before recording, and identities were anonymised during dataset preparation.

No metadata or personal participant details are included in this repository.

---

## 🏷️ File Naming

Audio files use anonymised names generated during corpus preparation to support consistency and participant privacy.
